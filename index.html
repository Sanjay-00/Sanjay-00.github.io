Multi-agent Pipeline · LangGraph
Blog Writing Agent
Multi-agent pipeline that mirrors how an experienced technical writer works. A Router classifies the topic and decides whether web research is needed. A Research node gathers Tavily results, deduplicates by URL, and produces a clean evidence pack. An Orchestrator plans each section with Pydantic-enforced contracts. Workers then write all sections in parallel via LangGraph Send, each grounded to verified citations only. A Merge node restores section order and assembles the final Markdown document.
⚡ Agentic routing · fan-out parallelism · schema-enforced section contracts
